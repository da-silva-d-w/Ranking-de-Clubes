Este web app foi feito com Streamlit para criar rankings de clubes de futebol.
Primeiramente, especifique os nomes dos campeonatos a serem utilizados como critério de grandeza e quantos pontos cada campeonato deve valer. 
Em seguida, basta informar quais clubes serão comparados e quantos campeonatos (entre os especificados anteriormente) foram vencidos por cada clube.
Ao final, será mostrada uma tabela com os clubes rankeados de acordo com suas pontuações.
